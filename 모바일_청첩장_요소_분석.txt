# 모바일 청첩장 필수 요소 분석 (보는 사람 관점)

## 조사 대상 사이트
- 디얼디어 (deardeer.kr)
- 프롬투데이 (fromtoday.co.kr)  
- 데어무드 (theirmood.com)
- 투아워게스트 (toourguest.com)
- Wix 청첩장 템플릿
- 바른손카드, 잇츠카드 등

## 1. 필수 요소 (100% 빈도)

### 💑 신랑신부 이름
- 한글명 (필수)
- 영문명 (선택적)
- 부모님 성함 포함
- 예: "김철수 ♥ 이영희"

### 📅 결혼식 정보  
- 날짜 (년.월.일, 요일 명시)
- 시간 (오전/오후 구분)
- 계절감 있는 표현
- 예: "2024년 10월 25일 토요일 오후 2시"

### 📍 장소 정보
- 예식장명 + 층/홀 정보
- 상세 주소
- 지도/네비게이션 연동 (카카오맵, 네이버지도, 티맵)
- 예: "서울웨딩홀 3층 그레이스홀"

### 📱 연락처
- 신랑 연락처 + 부모님
- 신부 연락처 + 부모님  
- 터치시 바로 전화 연결

## 2. 거의 필수 요소 (85-95% 빈도)

### 📸 커플 사진
- 메인 대표 사진 1-2장
- 갤러리 형태 (보통 3-12장)
- 고화질, 모바일 최적화 필수

### 💌 인사말
- 정중한 초대 메시지
- 개인적인 감사 인사
- 예: "저희 두 사람이 사랑의 결실을 맺게 되었습니다"

### 🎯 참석 여부 확인 (RSVP)
- 간단한 폼 (이름, 참석여부)
- 식사 여부, 동반자 수
- 개인정보 동의 체크

### 💰 계좌 정보
- 신랑측/신부측 축의금 계좌번호
- 원터치 복사 기능
- 은행명, 예금주명 명시

## 3. 자주 포함되는 요소 (60-80% 빈도)

### 🗓️ 달력/카운트다운
- D-Day 표시 (예: D-30)
- 달력에 결혼식 날짜 하이라이트
- 실시간 카운트다운

### 📝 방명록
- 축하 메시지 남기기 기능
- 실시간 확인 가능
- 관리자 승인 시스템

### 🚗 교통편 안내
- 지하철/버스 정보
- 주차장 위치 및 요금
- 셔틀버스 운행 정보

### 🎵 배경음악
- 자동재생/수동재생 선택
- 웨딩송 또는 의미있는 곡
- 음소거 버튼 필수

## 4. 선택적 요소 (30-50% 빈도)

### 📖 연애 스토리
- 만남부터 결혼까지 스토리
- 타임라인 형태 구성
- 사진과 함께 텍스트

### 🌸 화환 주문
- 온라인 화환 주문 링크
- 배송 정보 및 가격 안내
- 제휴 업체 연동

### 📺 라이브 스트리밍  
- 온라인 중계 링크
- 참석 못하는 분들을 위한 배려
- YouTube/Zoom 연동

### 🎁 웨딩 리스트
- 선물 리스트 연동
- 온라인 쇼핑몰 링크
- 신혼집 필요 물품

## 5. 트렌드/특별 요소 (10-30% 빈도)

### 🎬 영상 콘텐츠
- 프로포즈 영상
- 커플 인터뷰 영상
- 감사 영상 (부모님께)

### 📱 SNS 연동
- 인스타그램 해시태그
- 실시간 피드 연동
- 공유 기능 강화

### 🌍 다국어 지원
- 한/영 버전 제공
- 해외 하객을 위한 배려
- 번역 품질 중요

### ♿ 접근성 정보
- 휠체어 접근 가능 여부
- 엘리베이터/경사로 정보
- 시니어 친화적 디자인

## 개발 우선순위

### Phase 1 - MVP (최소 기능 제품)
1. 신랑신부 이름 + 부모님 성함
2. 결혼식 날짜/시간
3. 예식장 정보 + 지도
4. 연락처 정보
5. 기본 인사말
6. 메인 커플 사진 1장

### Phase 2 - 표준 버전
7. 사진 갤러리 (6-8장)
8. RSVP 참석 확인 폼
9. 축의금 계좌번호 + 복사기능
10. 교통편/주차 안내
11. 간단한 스크롤 애니메이션

### Phase 3 - 프리미엄 버전  
12. 배경음악
13. 방명록 기능
14. D-Day 카운트다운
15. 화환 주문 연동
16. 연애 스토리

## 기술적 고려사항

### 반응형 디자인
- 모든 모바일 기기 대응 (iPhone, Android)
- 세로/가로 모드 지원
- 다양한 화면 크기 최적화

### 성능 최적화
- 이미지 자동 압축 및 최적화
- 3초 이내 로딩 목표
- Progressive loading 적용

### 사용성 (UX)
- 원터치 전화 연결
- 간편한 공유 기능 (카카오톡, 문자)
- 시니어층도 쉽게 사용 가능한 UI

### 접근성
- 큰 폰트 사용
- 명확한 버튼 디자인
- 색상 대비 고려

### 공유 및 배포
- URL 단축 서비스
- QR코드 생성
- 소셜미디어 최적화 (Open Graph)

## 결론

모바일 청첩장에서 가장 중요한 것은 "필요한 정보를 빠르고 쉽게 찾을 수 있느냐"입니다.

핵심 성공 요소:
1. 필수 정보의 명확한 전달
2. 직관적이고 간단한 사용법  
3. 빠른 로딩 속도
4. 모든 연령층이 사용 가능한 접근성
5. 안정적인 공유 기능

Phase 1의 6가지 필수 요소를 완벽하게 구현한 후, 점진적으로 기능을 추가하는 것이 최적의 접근 방식입니다.
